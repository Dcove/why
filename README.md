# why
### 安装方法1：
```bash
# 需要git
cd ~
git clone https://github.com/pyg2007/why.git
cd why
bash install.sh
```
### 安装方法2：
```bash
# 需要wget
cd ~
wget https://raw.githubusercontent.com/pyg2007/why/master/why
cp why $PREFIX/bin
rm -rf why
chmod +x $PREFIX/bin/why
```
### 功能使用
BiliBiliHelper:  
等待代码跑完后，会自动进入编辑页面，使用i编辑，使用ESC+ZZ进行保存
